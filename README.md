# Node Parser: getTags
The getTags module (part of the upcoming Node Parser Package) recursively parses the DOM and output
the full CSS selection tags of each div into an array. for example

```css
"div#sr-header-area > div > div.drop-choices.srdrop > a:nth-child(41)"
"iframe#mv-single"
"div#mv-tiles"
"span#mv-noti-msg"
"span#mv-undo"
"span#mv-restore"
"div#mv-noti > div"
```
